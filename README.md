# بوت تيليغرام متعدد المهام

هذا البوت يعمل على Railway.

## المتغيرات البيئية المطلوبة

- `BOT_TOKEN`: توكن البوت من BotFather.
- `ADMIN_ID`: معرف الإدمن.

## متغيرات اختيارية

- `ENABLE_FILE_ANALYSIS`: لتفعيل تحليل الملفات (true/false). الافتراضي true.
- `OPENAI_API_KEY`: مفتاح OpenAI لاستخدام الذكاء الاصطناعي.

## النشر على Railway

1. ادفع الكود إلى مستودع GitHub.
2. أنشئ مشروعاً جديداً في Railway واربطه بالمستودع.
3. أضف المتغيرات البيئية في تبويب Variables.
4. سيتم بناء البوت تلقائياً باستخدام Dockerfile.
